# bad-apple

First run ``git clone https://www.github.com/Ubuntufanboy/bad-apple/``
Then ``pip3 install Pillow``
Then ``cd bad-aple``
Then ``chmod +x install.sh``
Then bash ./install.sh
Then ``python3 framer.py``
Then ``python3 convert.py``
Then ``python3 main.py``
